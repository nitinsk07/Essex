Launching into Cyber Security

Introduction to the existing world of Cyber Security

Learning outcomes

I will learn about:

  • Various Cyber Security roles and responsibilities
  • Ethics of the profession
  • Relevant computing architectures, applications and standards
  • Concepts of Cryptography, Penetration testing and ethical hacking
  • Programming concepts to facilitate security-based solutions to identified problems
  • Developing skills to implement secure information systems
  • The ethos of professional development
